Cara Menggunakan Audio Player:

1. Download lagu dari YouTube menggunakan yt-dlp atau tool lainnya
2. Simpan file audio dengan nama "lagu.mp3" di folder ini (audio/)
3. Atau ganti nama file audio sesuai keinginan, lalu update di video.html:
   - Ganti "audio/lagu.mp3" dengan "audio/nama-file-kamu.mp3"

Format audio yang didukung:
- MP3 (disarankan)
- OGG
- WAV

Tips:
- Gunakan kualitas audio yang bagus untuk pengalaman mendengarkan yang lebih baik
- Pastikan file audio tidak terlalu besar (disarankan di bawah 10MB)
